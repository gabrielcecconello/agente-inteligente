## **Agente Inteligente**
### **Introdução**
Este agente inteligente foi feito no contexto de uma avaliação para a disciplina de Sistemas Inteligentes de Software e Hardware. A tarefa consistia em criar um agente inteligente com Python e uma LLM, capaz:
- de receber e de processar um arquivo PDF (manual de um veículo),
- de aceitar comandos de voz e os converter em texto,
- de fazer uma pesquisa inteligente no manual com base nesses comandos,
- e de responder usando voz sintetizada.

### **Tecnologias Utilizadas**
Como mencionado, o agente foi desenvolvido em Python e a LLM utilizada foi a Gemini do Google. Optou-se por utilizar o TTS da Google por apresentar uma voz mais natural. Ademais, o desenvolvimento foi realizado com o Ubuntu, portanto se recomenda usar esse SO para o uso do agente.
